# Random Seed Rotator
Automatically rotates or randomizes Rust map seeds each wipe or restart.

## Features
- Chooses a random seed from your configured list
- Keeps world size fixed
- Works automatically on server restart
- Logs chosen seed in console

## Installation
1. Place `RandomSeedRotator.cs` into your `/oxide/plugins` folder
2. Restart your server or run `oxide.reload RandomSeedRotator`

## Configuration
Edit the seed list in the plugin file to match your preferred seeds.
